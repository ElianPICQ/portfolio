Projet Scolaire - Site de Présentation Wordpress

Sujet

Création d'un site web ayant pour but de présenter l'école (CFA INSTA) et les membres du groupe dont je fais parti avec le CMS WordPress.

Caractéristiques

  • Projet scolaire
  • Travail de groupe
  • Gestion de projet
  • WordPress

Description

Le but de ce projet était de se familiariser avec le CMS WordPress

Pour cela, il fallait reprendre le site de présentation fait précédement en HTML/CSS/JS et le reproduire sur WorPress.

Le site contient donc des cours et exercices réalisés pendant la foramtion ainsi que de courtes descriptions de l'école et des membres du groupe.
