Projet Scolaire - Site de Gestion des Pannes pour Orange

Sujet

Création d'un site web avec base de donnée en groupe.

Caractéristiques

  • Projet scolaire
  • Travail de groupe
  • Gestion de projet
  • Développement web
  • Gestion de base de données
  • Architecture MVC

Description

L'objectif de ce projet était de réaliser, en groupe, un site web connecté à une base de donnée mySQL / MariaDB.

Le site à été créé avec les langages HTML, CSS, JavaScript, PHP et MariaDB.

Il s'agit d'une plateforme sur laquelle pourraient se connecter clients et techniciens Orange servant à signaler les pannes survenues sur leurs appareils pour les uns et à prendre en charge ces pannes pour les autres.
